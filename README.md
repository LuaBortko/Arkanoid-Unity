# Arkanoid-Unity
Projeto desenvolvido na Unity como atividade de laboratório, consistindo na criação de um jogo simples inspirado no Arkanoid.

## Requisitos do Projeto

- O jogador controla uma nave espacial que se move horizontalmente na parte inferior da tela. 
- O objetivo é usar uma bola para destruir todos os blocos coloridos que estão localizados na parte superior da tela.
- Para começar o jogo, o jogador deve pressionar o botão de lançamento para soltar a bola. 
- A bola se move em uma trajetória reta até atingir um dos blocos ou a parede superior da tela. 
- Quando a bola atinge um bloco, ele é destruído e o jogador ganha pontos.
- Se a bola tocar na nave espacial, ela rebaterá e continuará em movimento. 
- Se a bola cair fora da tela, o jogador perde uma vida e deve começar novamente. 
- O jogo termina quando o jogador perde todas as suas vidas.
- Durante o jogo, o jogador pode coletar power-ups que oferecem habilidades especiais, como: 
    aumentar o tamanho da nave, 
    acelerar a bola ou 
    adicionar vidas extras. 
- À medida que o jogador avança pelos níveis, os blocos se tornam mais difíceis de destruir e os desafios aumentam.
